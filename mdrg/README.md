# MDRG 

This crate is used to describe the save file format used by the My Dystopian Robot Girlfriend game.

## .mdrg

The .mdrg file format is a simple json, but its quite obfuscated and difficult to understand at first sight.

In the code, everything (or almost) is well documented, so you can have a grasp of everything!
