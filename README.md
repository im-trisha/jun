# Jun

Jun is a GUI made to edit [My Dystopian Robot Girlfriend](https://incontinentcell.itch.io/factorial-omega) save files.


The program could still have some problems and maybe (just maybe) corrupt your save file, use with caution!

To install this program, check out the [installation](#installation) section

## Getting started

### Installation

Go to the [releases page](https://github.com/im-trisha/jun/releases/latest) and based on your platform:
- Windows: Download the .msi file and execute it
- Linux/MacOS: Run the curl command

### Building

To build this, you need a rust nightly compiler and run `cargo run`, as simple as that
